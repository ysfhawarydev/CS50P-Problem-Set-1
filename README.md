# 🐍 CS50 Python - Problem Set 1 Solutions

This repository contains my solutions for **Problem Set 1** from **CS50's Introduction to Programming with Python**.

The problem set focuses on conditional statements, logical operators, input validation, and decision-making in Python through a collection of practical programming challenges.

---

## 📚 Problems Included

### 🏦 Deep Thought

Respond correctly to the ultimate answer to life, the universe, and everything.

### 🍽️ Home Federal Savings Bank

Determine whether a user receives a greeting reward based on their input.

### 🧮 File Extensions

Identify the correct media type based on a file's extension.

### 💰 Math Interpreter

Perform basic arithmetic operations based on user input.

### 🍎 Meal Time

Determine whether a given time corresponds to breakfast, lunch, or dinner.
